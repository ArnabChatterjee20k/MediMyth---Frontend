# MediMyth - Frontend

## Routes

- Doctor Routes

    - Register a doctor

        ```
            /registration/doctor
        ```
    
    - Doctor Profile for Patients

        ```
            /doctor/:id
        ```
    
    - Doctor Profile for Doctor himself

        ```
            /doctor/profile
        ```
    
    - Viewing Schedules for Doctor

        ```
            /doctor/profile/
        ```
    
    - Viewing Appointemnts for Doctor

        ```
            /doctor/profile/appointemnt
        ```
